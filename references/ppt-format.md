# PPT 大纲页面输出格式模板

每页按以下格式输出，方括号内为替换说明。

---

## 页面输出格式

```
### PX — [类别]

**标题**：[结论先行的一句话标题，包含核心主张]

**关键字**：[关键词1] | [关键词2] | [关键词3] | [关键词4]

**核心内容**：
- 要点1：[小标题]——[详细说明，1-2句]
- 要点2：[小标题]——[详细说明，1-2句]
- 要点3：[小标题]——[详细说明，1-2句]
- 要点4：[小标题]——[详细说明，1-2句]
（每页4-6个要点）

**数据支撑**：[引用来源的数据或行业对比数据]

**视觉元素**：
- 主要图表：[推荐图表类型 + 简要描述]
- 配图建议：[图片场景推荐]
- 颜色基调：[主色 #hex] + [辅色 #hex]

**演讲要点**：[面向客户的演讲台词，2-4句，口语化]
```

## 示例

### P1 — 执行摘要

**标题**：以IMS-V6七产品协同体系，为客户构建智能制造数字基座

**关键字**：智能制造 | IMS-V6 | 七产品协同 | 业财一体化 | MOM平台

**核心内容**：
- 要点1：**客户战略定位**——[客户名称]是[行业]领军企业，产品承载[生态/标准]的品控使命。现有IT系统[现状描述]，亟需MES实现[核心目标]
- 要点2：**平台完整性**——IMS-V6是从底层BS工软平台(OS)到七产品全自研的国产MOM平台，BS+MES+QCS+EAM+WMS+IOT+STS原生协同——"出厂即协同"
- 要点3：**核心差异化**——BS统一底座提供低代码柔性 + IOT [数量]种已验证设备驱动 + STS支持[品牌列表]等ERP集成中枢
- 要点4：**分期策略**——一期聚焦[本期范围]→ 二期拓展[SMT/扩展]→ 三期[代工厂/自有工厂]，BS平台架构保证各期平滑衔接

**数据支撑**：[客户]2026年[行业排名]（来源：[来源]）；IMS-V6平台累计服务超500家制造企业

**视觉元素**：
- 主要图表：左侧客户品牌标识+产品渲染图 | 中部IMS-V6七产品图标环形排列 | 底部方案编号/日期/密级
- 配图建议：客户工厂/产品图 / IMS-V6产品架构图
- 颜色基调：科技蓝 #1A3A5C + 品牌色 + 白色背景

**演讲要点**：
尊敬的各位领导，感谢[客户]给予我们本次机会。我们深刻理解[客户]的独特身份——[核心定位]。今天呈现的方案，是基于对[客户]业务场景的深度理解，融合BS低代码平台、IOT设备驱动、STS多品牌ERP集成的一体化解决方案。IMS-V6不是拼凑的系统——从底层OS到七产品全自研，原生协同。

## 输出规范

1. 每页标题不可重复
2. 关键字 3-5 个，管道符分隔
3. 核心内容 4-6 个要点，每个要点格式为「**小标题**——详细说明」
4. 数据支撑必须标注数据来源或统计口径
5. 视觉元素必须包含主要图表类型+颜色基调（hex色值）
6. 演讲要点为面向客户的口语化台词
7. 所有颜色使用6位hex（如 `#1A3A5C`），不用简称（如 "blue"）
